## Overview

All datasets and analysis code associated with my first-authored publications are hosted openly on the **Open Science Framework (OSF)**. This includes raw data, preprocessing scripts, and the statistical analyses used in each paper.

<Callout>
  Open science practices — pre-registration, open data, and open code — are central to how I approach research. If you use any of these datasets in your own work, feel free to reach out.
</Callout>

## What's included

Each project on OSF corresponds to a published or in-preparation manuscript, and contains:

- **Raw data** — trial-level movement and response data
- **Preprocessing scripts** — R or Python code to clean and transform data
- **Analysis code** — statistical models, figures, and supplementary analyses
- **README** — description of variables, file structure, and how to reproduce results
